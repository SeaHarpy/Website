Neural Network

Made with:

This is a Neural Network that can be used to identify cats and dogs. I did not use any PyTorch or TensorFlow to create this network. It uses a linear regression model to predict the output of a given input. The network is trained using the backpropagation algorithm, and it able to predict if the given image is a cat or a dog. It could also be trained using a different dataset to predict other objects.

Youtube Video Generator

Made with:

This project allowed me to explore how to prompt different AI models including GROQ, and OpenAI's GPT-4. I learned prompt engineering techniques in order to recieve valid JSON responses.The project prompts OpenAI/Groq with a topic chosen by the user and the model will generate facts that Edge-tts will convert to audio. Meanwhile there is a thread that goes to Pexels and queries their API for images realated to the topic. The images are then combined into a video with the audio. The next step for this project is to use Selenium to automatically uploaded the videos to a social media site.
